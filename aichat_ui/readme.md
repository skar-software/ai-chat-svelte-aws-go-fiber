# AI Chat Widget

A professional, reusable AI chat interface built with **Svelte 5** and **Tailwind CSS**. Designed for seamless embedding into SaaS applications, providing a rich, interactive assistant experience.

## ✨ Features

- **Svelte 5 Powered**: Leverages the latest Svelte features for high performance and clean code.
- **Rich Elements**: Supports specialized AI response types:
  - **Code Blocks**: Syntax highlighting via Shiki with light/dark mode support.
  - **Plans**: Interactive, collapsible deployment/action plans.
  - **Artifacts**: Display complex data or documents generated by the AI.
  - **Tasks/Queue**: Real-time progress tracking for long-running operations.
  - **Citations**: Inline references for grounding AI responses in data.
- **Theme Awareness**: Automatically detects and reflects the parent application's theme (Light/Dark).
- **Responsive Design**: Fluid layout that works on desktop and mobile.
- **Easy Integration**: Drop-in component with simple configuration for API base URL, tenant context, and authentication.

## 🏗️ Technical Architecture

The widget is divided into a layered structure for maximum maintainability:

### 1. `ChatWidget.svelte` (Root Component)
The main entry point. It manages:
- **Navigation & Sidebar**: Layout for multi-conversation support.
- **Theme Detection**: Observes `html.dark` class to sync its internal styling.
- **API Interaction**: (Demo Mode) Simulates or handles communication with the AI backend.
- **State Management**: Uses Svelte 5 `$state` and `$derived` for reactive message flow.

### 2. `ai-elements/` (Core UI Library)
A collection of atomic and molecular components used to build the chat experience:
- **`conversation/`**: Wrappers for the message scroll area and empty states.
- **`message/`**: The standard message bubbles (User vs. Assistant) with different variants (`contained`, `flat`).
- **`code/`**: Advanced code rendering with line numbers and copy functionality.
- **`plan/`**: Structured step-by-step guides.
- **`prompt-input/`**: Rich text input area with auto-expansion and character limits.
- **`loader/`**: Smooth animations for "AI is thinking" states.

### 3. `ChatMessageItem.svelte`
The dispatcher component. It analyzes each part of an AI's response (text, code, plan, etc.) and renders the appropriate `ai-element`.

## 🚀 Getting Started

### Integration

To add the chat widget to your page:

```svelte
<script>
  import ChatWidget from '$lib/components/ChatWidget.svelte';
</script>

<ChatWidget 
  apiBaseUrl="https://api.yourserver.com/v1/chat"
  tenantId="your-tenant-id"
  title="My AI Assistant"
/>
```

### Theme Support
The widget follows the `dark` class on the `<html>` element by default. If your app uses a different method, you can manually override the theme using the `mode` prop:

```svelte
<ChatWidget mode="dark" />
```
